# virtual_girlfriend_bot

## 中文简介

巴西留子用 Perl 写的 Trollgrad Bot，专门爬大学教务处 PDF 动态发到 Telegram 群，专门拿来整蛊朋友。名字是 Troll + Prograd 的谐音，代码里还写着 'just to mess up with Cardu'，笑死。虽然描述写的 virtual girlfriend 是假的，但这个整蛊精神很到位。

## 有趣角度

这个 repo 名字叫 'Trollgrad Bot'，是 Troll（恶搞）+ Prograd（巴西大学教务处）的谐音梗。代码注释里写着 'Perl just to mess up with Cardu'，明摆着就是拿来整蛊朋友的。实际功能是爬大学 PDF 通知发送到 Telegram 频道，虽然描述里写的 'virtual girlfriend' 是骗人的，但 'Trollgrad' 这个谐音梗确实有内味。

## 小红书内容

**救命！我被一个 Perl 脚本整蛊了整整一个学期 😂😂**

---

事情是这样的——

我在巴西留学的朋友给我安利了一个 GitHub 项目，叫 **virtual_girlfriend_bot** 🤖✨

名字听着是不是还挺正经的？什么虚拟女友机器人~

但我打开一看——

**"Trollgrad Bot"** 💀

Troll + Prograd = Trollgrad？？？

这谐音梗谁教的啊哈哈哈哈 😂😂😂

而且代码里还明晃晃写着：

> "Perl just to mess up with Cardu"

意思是"Perl 就是来搞 Cardu 的"——Cardu 是他们学校教务处的名字！

所以这个 bot 到底是干嘛的？

其实就是 **爬大学教务处的 PDF 通知，然后自动发到 Telegram 群** 📄➡️📱

BUT！它的真实目的根本不是什么"虚拟女友"👋

是专门拿来 **整蛊朋友** 的哈哈哈哈 🤣🤣

想象一下——

你朋友以为你在认真做课业追踪，结果 bot 半夜把教务处更新全搬过来了

你朋友以为收到的是"今日作业提醒"，结果打开是"期末考试安排"😱

这波操作，我只能说——

**程序员浪漫，真的不是一般人能懂的** 💕

**整蛊精髓：**
- 用 Perl 写（明摆着就是故意的）
- 专门爬 PDF 发 Telegram
- 代码注释写"就是为了搞 Cardu"

笑死，这大概是我见过最"坦诚"的整蛊项目了 😂
