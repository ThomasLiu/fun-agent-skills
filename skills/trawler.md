# trawler

## 中文简介

一个 Express 中间件，专门 troll 爬虫黑客。扫描 phpMyAdmin、wp-login.php 的爬虫统统给你301重定向到 Rickroll 视频（你懂的），顺便清理日志。名字谐音梗：trawler = trolling + crawler，拖网渔船还能 catch 'em all。接上去之后你的服务器就变成爬虫陷阱了 🪤

## 有趣角度

专门给爬虫下套的恶搞中间件！扫描 phpMyAdmin、wp-login.php 的爬虫都会被301重定向到 Rickroll 视频（经典整蛊链接），还自带「 Gotta catch 'em all」宝可梦梗。名字也很有趣：trawler = trolling + crawler（拖网渔船还能捕一堆鱼）。本质是：用你的服务器带宽去整蛊爬虫，优雅地让黑客/爬虫去看 Rick Astley 😂

## 🎯 它是干啥的？

就是一行中间件的事儿 —— 装上之后，你的服务器瞬间变成 **爬虫陷阱 🪤**

当黑客/爬虫满怀信心地扫描你的 `phpMyAdmin`、`wp-login.php`，想要搞事情的时候...

👉 **啪一下！直接 301 重定向到 Rickroll 视频！**

就是那个 🎵 *"Never Gonna Give You Up"* 🎵 的 Rick Astley！

黑客：我黑进去了！
服务器：你被 Rickroll 了 😂

## 🌊 名字也超有梗

`trawler` = **trolling + crawler**（整蛊爬虫 + 拖网渔船 🌊）

寓意就是：你的服务器像拖网渔船一样，一网打尽所有不请自来的爬虫！

还自带宝可梦梗：**Gotta catch 'em all!** 只不过 catch 的是爬虫不是皮卡丘 ⚡

## 😂 为啥我觉得它绝了

- **用黑客的带宽去看 Rick Astley** —— 这波以彼之道还施彼身，优雅！
- **自动清理日志** —— 不留痕迹，优雅离场
- **纯 Node.js + Express** —— 装起来就一行代码

想象一下：黑客对着你服务器一顿操作，结果发现自己看了 10 遍 Rick Astley... 这心理阴影面积得有多大 😂

## 🛠 咋用

```js
const trawler = require('trawler');
app.use(trawler());
```
